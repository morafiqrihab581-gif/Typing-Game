# ⌨️ Typing Speed Test Game

A simple and interactive **Typing Speed Test** built with **HTML, CSS, and JavaScript**.

Practice typing full sentences while tracking:

- 🚀 Live **WPM (Words Per Minute)**
- 🎯 **Accuracy**
- ✅ Correct characters (green)
- ❌ Incorrect characters (red)
- ⏳ Countdown timer

---

## 📸 Features

- 🎲 Random sentence generator
- ⚡ Real-time typing validation
- 📈 Live WPM calculation
- 🎯 Accuracy tracking
- 🔄 Restart game button
- 🌙 Responsive dark UI
- ✍️ Character highlighting system

---

## 🎮 How to Play

1. Start typing the displayed sentence  
2. ✅ Correct letters turn **green**  
3. ❌ Incorrect letters turn **red**  
4. Finish the sentence before time runs out ⏳  
5. Try to achieve the **highest WPM** 🚀

---

## 📊 WPM Formula

```text
WPM = (Correct Characters / 5) / Time in Minutes
```
